# chrome-extension-word-highlighter

特定Webページ内の特定の文言をハイライト表示するChrome拡張機能（Manifest V3対応）です。  
JavaScript / Chrome APIの学習を目的として段階的に開発しています。

---

## 機能一覧

- [x] **Step 1:** 特定ページの固定ワードをハイライト
- [ ] **Step 2:** 動的コンテンツ（SPA / スクロール追加）の監視・ハイライト対応
- [ ] **Step 3:** ポップアップUIからのキーワード入力・変更
- [ ] **Step 4:** `chrome.storage` による設定保存 ＆ 正規表現・カラー変更対応
